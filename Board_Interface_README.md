# Professional Case Board Interface

The board interface provides a clean, professional view of active cases designed for sharing with colleagues and clients.

## Accessing the Board

**URL for colleagues on your WiFi network:**
`http://192.168.0.98:8000/board`

## Features

### Professional Design
- Clean, modern interface without distracting elements
- No videos or sports content
- Professional header with your branding
- Live status indicator showing real-time updates

### Case Information Display
- **Client Name** - Full client names with priority indicators
- **Case Name/Number** - Case titles and reference numbers
- **Type** - Case category (MVA, Medical Malpractice, etc.)
- **Stage** - Current litigation stage
- **Status** - Case status with color-coded badges
- **Current Focus** - Latest focus/task for each case
- **County** - Jurisdiction information
- **Paralegal** - Assigned paralegal

### Smart Organization
- **Top Priority cases** highlighted in gold with star indicators
- **Needs Attention cases** highlighted in amber
- **Waiting cases** subtly marked
- Alphabetical sorting within priority groups
- Excludes archived cases automatically

### Live Updates
- **Real-time data** updates every minute
- **Live indicator** shows connection status (● LIVE / ● OFFLINE)
- **Progress bar** shows time until next update
- **Current date/time** display in header

## Logo Requirements

**PNG File Name:** `Professional_Logo.png`

**Location:** `static/pngs/Professional_Logo.png`

Please save your professional logo with this exact filename in the `static/pngs/` folder. The logo should be:
- High resolution (300+ DPI recommended)
- Horizontal layout preferred
- White/light colored for use on dark background
- PNG format for transparency support

## Usage Tips

### For Colleagues
- Share the URL: `http://192.168.0.98:8000/board`
- Works on any device connected to your WiFi
- Responsive design works on tablets, phones, laptops
- No login required - read-only access

### For Presentations
- Full-screen friendly design
- Professional appearance suitable for client meetings
- Live data ensures information is always current
- Clean typography optimized for readability

### For Office Displays
- Can be displayed on wall-mounted monitors
- Auto-updating content requires no maintenance
- Professional appearance maintains office aesthetics

## Security Notes

- Only accessible within your WiFi network
- Read-only access - no case data can be modified
- No sensitive financial or detailed case information displayed
- Cases marked as archived are automatically excluded

## Browser Compatibility

Works with all modern browsers:
- Chrome, Firefox, Safari, Edge
- Mobile browsers on iOS and Android
- Smart TV browsers

The interface automatically adapts to different screen sizes and orientations.
